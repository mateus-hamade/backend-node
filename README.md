# API node 🐦

### Projeto ⚒️
- O objetivo deste projeto é criar uma API usando Nodejs, Prisma e Jest.

### Ferramentas utilizadas ⛏️
- Prisma (ORM)
- Jest (Tests)
- Node

### Executando o código
- Instalando as dependencias 📚

```
>> npm install
```

- Executando o código 🚀
```
>> node server
```
- Executando os testes 🥳
````
>> npm test
````
